Mode d'emploi SoyezCreateurs

]

Ce squelette est entièrement personnalisable et il va vous permettre de changer les différents critères suivants :

  • Création et modification des mots clés
  • Pour changer la page d’accueil
  • Pour changer le nom du site et son logo
  • Configuration du Squelette SoyezCreateurs (ex : position Logo) [1]
  • Choisir la disposition des différents parties du site parmi 40 modèles disponibles (source : Layoutgala)

Vous trouverez aussi plusieurs documentations disponibles aux adresses ci-dessous :

  • pyrat.net :
    • Rédacteurs SPIP : la boîte à outils
    • Les raccourcis typographiques indispensables de SPIP
    • Rédiger pour Internet afin d’être lu et trouvé
    • Erreurs classiques de mise en forme typographique
  • Les 2 groupes de Mots Clés :
    • Navigation
    • Fonctionnels
Pour retrouver ce mode d’emploi, consulter le site de l’auteur.

[1] Attention, l’accès à cette configuration est réservée aux webmestres du site, par défaut, l’auteur n°1 avec ce squelette.
